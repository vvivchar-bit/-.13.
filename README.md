## Задача 2.3 — гістограма відносних частот

У задачі 2.3 задано інтервальний варіаційний ряд.

Було обчислено:

- загальну кількість спостережень `n`;
- відносні частоти `pi`;
- ширину інтервалу `h`;
- висоти гістограми `f*`.

Формули:

```text
pi = ni / n
h = права межа - ліва межа
f* = pi / h
```

Для заданого ряду отримано:

```text
n = 100
Σpi = 1
```

Також побудовано гістограму відносних частот.

## Задача 3.2 — вибіркове середнє та дисперсія

У задачі 3.2 використано інтервальний варіаційний ряд із задачі 2.3.

Було знайдено:

- середини інтервалів `xi`;
- добутки `ni·xi`;
- квадрати середин інтервалів `xi²`;
- добутки `ni·xi²`;
- вибіркове середнє;
- вибіркову дисперсію.

Формули:

```text
x̄ = Σ(ni · xi) / n
Dв = Σ(ni · xi²) / n − x̄²
```

Отримано:

```text
n = 100
Σ(ni · xi) = 940
Σ(ni · xi²) = 9540
x̄ = 9.4
Dв = 7.04
```

## Задача 4.3 — надійний інтервал для математичного сподівання

У задачі 4.3 потрібно побудувати надійний інтервал для математичного сподівання.

Вхідні дані:

```text
σ = 4.5
x̄ = 15.6
n = 100
γ = 0.99
```

Для надійності `γ = 0.99` використано:

```text
zγ = 2.576
```

Формула похибки:

```text
Δ = zγ · σ / √n
```

Обчислення:

```text
Δ = 2.576 · 4.5 / √100 = 1.1592
```

Надійний інтервал:

```text
a ∈ (x̄ − Δ; x̄ + Δ)
a ∈ (15.6 − 1.1592; 15.6 + 1.1592)
a ∈ (14.4408; 16.7592)
```
